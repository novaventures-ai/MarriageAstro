## What is Astro Marriage?

Astro Marriage is a free web application that provides comprehensive Vedic astrology marriage analysis:

- **Ashtakoot Milan (36-point scoring)** — Evaluates 8 Kootas: Varna, Vashya, Tara, Yoni, Graha Maitri, Gana, Bhakoot, Nadi
- **Marriage Timing Prediction** — Multi-system convergence using Vimshottari Dasha, Jupiter-Saturn transits, KP sub-lords, and Jaimini Chara Dasha
- **Spouse Characteristics** — Predicted via 7th house lord, Darakaraka (Jaimini), and Upapada Lagna
- **Mangal Dosha Analysis** — Triple-check (Lagna, Moon, Venus) with 15+ cancellation rules
- **Personalized Remedies** — Gemstones, Lal Kitab remedies, and mantras

| System | Purpose |
